Webview Kiosk — это бесплатное приложение с открытым исходным кодом для Android, предназначенное для безопасного веб-сёрфинга в режиме киоска.

Приложение создано для малого бизнеса и тех владельцев устройств, которым нужно простое и автономное решение для киоскового режима.

<b>Основные возможности</b>

- <b>Режим закрепления (Lock Task Mode / Pin)</b>: предотвращает доступ к главному экрану устройства (рабочему столу), другим приложениям и строке состояния.
- <b>Защищённые настройки</b>: все параметры конфигурации защищены биометрией или учётными данными устройства.
- <b>Фильтрация URL</b>: возможность использования регулярных выражений для ограничения доступа к веб-ресурсам через чёрные и белые списки.
- <b>Экспорт и импорт</b>: создание резервных копий пользовательских настроек и конфигураций, и восстановление их из файлов бэкапов.
- <b>Локальные файлы</b>: возможность работы в офлайне с локальными файлами, будь то изображения, аудио, видео или HTML-файл.
- <b>Управляемые конфигурации</b>: на корпоративных устройствах настройки можно задавать и ограничивать удалённо через сертифицированную систему MDM/EMM.

<b>Дополнительные разрешения</b>

Приведённые ниже разрешения Android <b>необязательны</b>, и соответствующие функции <b>по умолчанию отключены</b> в настройках приложения:

- <code>CAMERA</code>
- <code>RECORD_AUDIO</code>
- <code>ACCESS_FINE_LOCATION</code>
- <code>ACCESS_COARSE_LOCATION</code>

Активируйте их только в том случае, если они необходимы вашему веб-сайту или веб-приложению. Эти разрешения не будут использоваться для других целей.
