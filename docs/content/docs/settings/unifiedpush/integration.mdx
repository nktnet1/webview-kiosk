## 1. Curl (ntfy)

If you use ntfy as your distributor and push server, it is possible
to use curl commands to send push messages directly to {APP_NAME}.

As the payload is unencrypted, you will need to enable
`UnifiedPush -> Process Unecnrypted Messages`.

Example:

```bash tab="Command"
# NOTE: Replace the URL with your registered endpoint
curl 'https://ntfy.sh/upKsJhj5FnOYoT?up=1' \
  -d '{
    "type": "command",
    "command": "go_to_url",
    "data": {
      "url": "https://webviewkiosk.nktnet.uk"
    }
  }'
```

```bash tab="Settings"
# NOTE: Replace the URL with your registered endpoint
curl 'https://ntfy.sh/upKsJhj5FnOYoT?up=1' \
  -d '{
    "type": "settings",
    "data": {
      "settings": {
        "appearance.theme": "LIGHT"
      }
    }
  }'
```
